**NAS management for everyone.** Mixed drive sizes. Docker orchestration. Single binary. No lock-in.

## Editions

### Community Edition (CE) — Free

Open source Docker container management for homelabs.

- 157 pre-configured apps across 12 categories with one-click deployment
- Real-time container monitoring
- Community app store
- MIT License

**[View on GitHub](https://github.com/smashingtags/homelabarr-ce)** (v1.1.0 — live and deployed)

---

### Professional Edition (PE) — From $39

Full NAS management in a single Go binary with an embedded React dashboard.

- SnapRAID + MergerFS (mixed drive sizes — the Unraid killer feature)
- ZFS pool management
- Docker container orchestration
- File sharing (SMB/NFS) via web UI
- Network management with Cloudflare DNS
- Real-time monitoring (CPU, RAM, disk, SMART)
- Cache mover with intelligent tiering

## History

HomelabARR started as a community project born from the PGBlitz/PlexGuide era — rescuing documentation and configurations after the original projects collapsed. Read the full story: [Why I Built HomelabARR](https://mjashley.com/writing/why-i-built-homelabarr/).

The original codebase is preserved at [homelabarr-legacy](https://github.com/smashingtags/homelabarr-legacy) (archived).
